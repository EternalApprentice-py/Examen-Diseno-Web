# Examen-Diseno-Web

Template con integración y adaptación de elementos del Grupo 5

Este proyecto se desarrolla a partir de un template seleccionado y aprobado por la clienta, de acuerdo con las sugerencias iniciales planteadas.
El template utilizado es el siguiente:
(https://themewagon.com/themes/free-html5-bootstrap-5-business-website-template-carserv/
)

En evaluaciones previas se trabajaron los mockups correspondientes a la estructura, estilos y acciones de interacción, lo que permitió orientar la adaptación del diseño final.

En términos generales, la finalidad del sitio es ofrecer una navegación clara y cómoda para el usuario final, con acciones atractivas pero no invasivas, textos legibles y acceso sencillo a la información clave. En este caso, se priorizó la visualización de los modelos de vehículos asociados al negocio.

Se utilizaron imágenes entregadas por la clienta, además de algunas extraídas desde sus redes sociales. También se incorporaron mensajes y textos proporcionados por la propia clienta, complementados con otros desarrollados por nosotros como referencia, los cuales pueden ser ajustados posteriormente según las necesidades del proyecto.

Por último, la estructura principal del template se mantiene en el formato clásico:

img/
css/styles.css
js/main.js y mejora.js
index.html
